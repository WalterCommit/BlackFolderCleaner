# Folder Cleaner / Klasör Temizleyici

**English | Türkçe**

---

## Overview / Genel Bakış

**English:**  
Folder Cleaner is a lightweight Python GUI application designed to permanently delete all files and subfolders inside a selected directory. It is perfect for cleaning test directories, temporary files, or any folder you no longer need. The application features a simple interface, live logging, and adjustable deletion speed for safety and convenience.  

**Türkçe:**  
Klasör Temizleyici, seçilen bir klasör içindeki tüm dosya ve alt klasörleri kalıcı olarak silmek için geliştirilmiş hafif bir Python GUI uygulamasıdır. Test klasörlerini, geçici dosyaları veya artık ihtiyaç duymadığınız klasörleri temizlemek için idealdir. Uygulama basit bir arayüz, anlık işlem günlüğü ve güvenlik için ayarlanabilir silme hızı sunar.

---

## Features / Özellikler

**English:**  
- Select any folder through a graphical interface.  
- Delete all files and subfolders permanently.  
- Adjustable deletion speed (time in seconds between deleting items).  
- Real-time log of actions performed.  
- Terminal-style black background with green text for easy readability.  
- Designed for Python 3.7 (32-bit) — lightweight and portable.  
- Simple, user-friendly GUI suitable for beginners and advanced users alike.  

**Türkçe:**  
- GUI üzerinden herhangi bir klasörü seçebilme.  
- Tüm dosya ve alt klasörleri kalıcı olarak silme.  
- Silme hızını ayarlayabilme (her dosya/klasör arasında bekleme süresi).
- Yapılan işlemlerin anlık günlüğünü görüntüleme.  
- Siyah arka plan ve yeşil yazılarla terminal tarzı görünüm.  
- Python 3.7 (32-bit) uyumlu, hafif ve taşınabilir.  
- Basit ve kullanıcı dostu arayüz, hem acemiler hem de ileri düzey kullanıcılar için uygun.
